# UAV大飛機-航電組開發流程 3

- 本次目標:
    - 了解offboard模式如何控制
    - 編輯並編譯，利用Gazebo自行測試編譯結果

## 找到範例檔案並編輯─單點飛行

```bash
cd ~/ws_offboard_control
grep -R "msg.position" src/px4_ros_com -n
```

應該會找到
```bash
src/px4_ros_com/src/examples/offboard/offboard_control.cpp
```

打開
```bash
code src/px4_ros_com/src/examples/offboard/offboard_control.cpp
```

改成(範例)
```C++
msg.position = {5.0, 0.0, -5.0};
```

再重新編譯
```bash
cd ~/ws_offboard_control
source /opt/ros/jazzy/setup.bash
colcon build --packages-select px4_ros_com
source install/local_setup.bash
ros2 run px4_ros_com offboard_control
```
