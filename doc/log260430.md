## offboard mode 結構
檔名:src/examples/offbaord/offboard_control.cpp
類別:class OffboardControl: public rclcpp::Node
main()裡面只做rclcpp::init()、spin()、shutdown()

## node 介紹
三個publisher:

- /fmu/in/offboard_control_mode
- /fmu/in/trajectory_setpoint
- /fmu/in/vehicle_command

重要成員

- timer_
- offboard_setpoint_counter_

## 建構子OffboardControl()再做什麼

- 建立三個publisher
- 把offboard_setpoint_counter_設成0
- 建立一個100 ms timer

## timer callback
流程幾乎都在callback裡
```C++
auto timer_callback = [this]() - > void {
    if (offboard_setpoint_counter_ == 10) {
        publish_vehicle_command(...切到 offboard...);
        arm();
    }
    publish_offboard_control_mode();
    publish_trajectory_setpoint();

    if (offboard_setpoint_counter_ < 11) {
        offboard_setpoint_counter_++;
    }
}
```
